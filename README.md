# 🧪 A/B Testing Analysis for Conversion Optimization

## 📌 Overview
This project analyzes the results of an A/B test designed to evaluate changes in user conversion behavior. The analysis focuses on comparing control and experimental groups to determine whether the new variation leads to a statistically significant improvement.

## 🎯 Objective
To assess the impact of a product or design change on user conversion rates and determine whether the new version should be implemented.

## 🛠️ Tools & Technologies
- Python
- pandas
- scipy / statsmodels
- Data Cleaning
- Statistical Analysis

## 📂 Dataset
The dataset includes user interaction data from an A/B test experiment:
- User ID
- Group assignment (control vs treatment)
- Conversion events
- Timestamps

## 🔍 Key Steps
- Cleaned and validated experiment data
- Verified group distribution and sample sizes
- Checked data consistency and removed anomalies
- Calculated conversion rates for each group
- Performed statistical hypothesis testing (proportions test)
- Evaluated statistical significance using p-values
- Interpreted results for business decision-making

## 📈 Key Findings
- Conversion rates differed between control and test groups
- Statistical testing determined whether the difference was significant
- Results provided clear evidence to support or reject the new version

## 💡 Business Impact
This type of analysis enables companies to:
- Make data-driven product decisions
- Reduce risk when implementing changes
- Optimize conversion rates
- Improve user experience based on evidence
- Validate product hypotheses before scaling
